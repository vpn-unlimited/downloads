常见问题。

什么是 Microsoft Windows 的 VPN 客户端？

VPN 是 Virtual Private Network（虚拟专用网络）的缩写。VPN 软件的主要目标是加密您在 Windows 上的流量并保障在线隐私。借助桌面 VPN 应用，您可以免受监控、增强对网络威胁的防护，并畅享无边界的互联网！

适合 Windows 桌面的 VPN 应具备哪些条件？

优秀的 Windows VPN 应当结合强大的 AES-256 加密、稳定的原生应用、广泛的服务器网络以及清晰的隐私控制。Windows 版 VPN Unlimited 全部满足：Kill Switch、专属流媒体服务器、最佳服务器自动选择，以及严格的无日志政策。

如何在台式机上获取并设置 VPN？

从 Microsoft Store 或我们的网站下载 Windows 版 VPN Unlimited。运行安装程序，按屏幕提示完成安装。打开应用，使用 KeepSolid ID 登录，选择服务器位置，点击“开始”。

如何在 Windows 上使用 VPN？

在 Windows PC 上打开 VPN Unlimited 应用，选择服务器：通过地图按区域选择、按 ping 在最佳服务器列表中选择，或按专用类别（流媒体、BT、额外安全）选择。点击“开始” — 您的 IP 即被隐藏、流量被加密。
